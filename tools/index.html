You Rush Local Debug Tools

Offline browser tools for checking layout, sprites, hitboxes, obstacles, and boss visuals before compiling or installing the Android app.

Offline Debug Workbench Best first stop for sprite crops, frame components, foot/contact lines, runtime strips, and gameplay composition snapshots. Sprite Sheet Audit Frame-by-frame PNG inspection with checkerboard transparency, frame boxes, crop overlays, labels, and grid. Gameplay Preview HUD, overlays, controls, hitboxes, contrast, device presets, and 15 px layout grid. Gear + Obstacles Driftwood logs, river logs, icebergs, snow piles, snowballs, bear spray, debug badges, and hitboxes. Laser Eyes Preview Polar bear beam origin, sweep line, warning reticle, and boss framing checks. Midnight Sun Lasers Twin-eye sun beams, layered glow, attack sweep, hitbox, and runner scale/speed checks. Menu Preview Main menu layout, badge placement, buttons, and first-screen readability checks. Debug Tuning Dashboard Quick reference for reporting numbered DEBUG screenshots during phone/emulator testing.
