You Rush 3.31 Debug Tuning Dashboard

Use this alongside DEBUG mode screenshots. Turn DEBUG on in the game, capture the numbered object, then report the number, type letter, and what feels wrong.

Preview Screens

  • Gameplay preview checks HUD, panels, controls, and debug hitboxes.
  • Laser eyes preview checks boss beam origin, sweep, and hitbox.
  • Gear + obstacle preview checks logs, snow piles, icebergs, snowballs, and bear spray.
  • Sprite sheet audit checks actual PNG frames for transparency and crop artifacts.

Report Format

  • 3G obstacle sprite/hitbox issue.
  • 2H wildlife sprite, motion, or collision issue.
  • 1B boss pose, contact, tell, or hitbox issue.
  • 4A boss attack issue such as beam/projectile fairness.
  • 5U pickup readability or placement issue.

3.31 Sprite Checks

  • River logs should read as wet logs, not racks.
  • Snowbanks should look like grounded snow piles.
  • Icebergs should be jagged and blue-white.
  • Antler barricades should read as antlers lashed together.
  • Vector sprite edges should have no PNG fringe.
  • Use the sprite-sheet audit frame number for PNG crop/artifact reports.

Bear Spray Checks

  • SPRAY pickup should look like an orange can.
  • Hold FIRE should spray from the runner's forward hand.
  • The cone should be short-range and readable.
  • Spray should stun close wildlife, not replace snowballs.
  • Spray should interrupt a boss lunge only when close.

Laser Beam Checks

  • Beam starts at the boss eyes, not chest or paws.
  • Beam sweeps up/down toward the runner.
  • Player can dodge by jumping or repositioning.
  • DEBUG attack label should read BEAM DODGE.
  • Hitbox should be thinner than the glow.

Growth Hooks

  • Gear can become unlockable and upgradable.
  • Obstacle sprites support themed level packs.
  • Trail Passport can reward cosmetics and gear skins.
  • Keep paid growth cosmetic/convenience-first, not pay-to-win.
  • Use Daily Rush and badges for retention loops.
