# You Rush 4.1.2 Quick Voice Notes

Version 4.1.2 reduces field review to one action: tap `MIC`, speak, and resume.

## Build Identity

```text
versionName: 4.1.2
versionCode: 412
badge: ALASKA PASSPORT v4.12
debug APK: app/build/outputs/apk/debug/you-rush-alaska-4.1.2-412-debug.apk
```

## Walking Workflow

1. Tap `MIC` when something feels wrong.
2. The exact frame freezes and Android speech recognition opens.
3. Speak naturally; the first transcription is saved automatically.
4. Gameplay resumes with stage, encounter, seed, progress, and visible IDs
   already attached to the note.

`NOTE` remains available for deliberate typing and copying the complete log.
Its compact editor hides raw technical context so the form contains only the
text field, priority checkbox, and actions. If speech recognition is missing,
the game falls back to this editor automatically.
