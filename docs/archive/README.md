# Documentation Archive

This archive preserves historical release notes, milestone notes, and rapid
iteration logs from the Alaska beta.

These files are intentionally not the first stop for current development. Use
the root `README.md` and `docs/README.md` for active project status, build
instructions, QA, versioning, and launch-readiness docs.

Archived notes are retained for traceability: they explain why older gameplay,
sprite, boss, splash-screen, and debug-tool changes were made.
