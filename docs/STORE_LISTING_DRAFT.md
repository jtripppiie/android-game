## App name

You Rush

## Short description

A personalized Alaska arcade runner starring your own face.

## Full description

You Rush is a fast, funny, local-first arcade game by TripperDeeLabs.

Pick an Alaska stage, add your face if you want, and survive a ridiculous local arcade challenge. Move, jump, throw snowballs, use trees carefully, chase combos, and try to clear each stage.

The Alaska beta includes five stages, personalized character art, stage intros, near-miss scoring, combo streaks, incoming callouts, tree timing, snowball interactions, stage challenges, and local saved progress.

No account is required for the current build.

## Feature bullets

- Add your own face to the player character
- Five Alaska stages
- Arcade movement and touch controls
- Snowball interactions
- Tree timing mechanic
- Near-miss bonus scoring
- Combo streaks
- Stage intro overlays
- Incoming object callouts
- Local progress and best score
