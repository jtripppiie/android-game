# You Rush 4.2.2 Compact Notes

The text review workflow no longer opens standard dialog chrome or forces the
software keyboard over the captured game frame.

- The composer is a 112 dp top-edge strip with no background dimming.
- Landscape width is capped at 38%/390 dp; portrait uses 92% width.
- The keyboard stays closed until the reviewer taps the single-line field.
- Scene context and visible debug IDs remain attached automatically.
- `FIX FIRST`, `SAVE`, and `CANCEL` remain available in the strip.
- Tapping outside cancels and returns to the paused frame.

Voice notes remain the fastest zero-keyboard capture path through `MIC`.
